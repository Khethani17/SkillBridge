# SkillBridge

SkillBridge is a platform where people can **exchange skills instead of paying money**.

Users can offer skills they already have and connect with others who want to learn them. In return, they can learn new skills from other users.

The goal is to create a community where people help each other grow by **trading knowledge and experience**.

---

## Examples of Skill Exchange

* Cooking ↔ Fitness training
* Photography ↔ Video editing
* Web development ↔ Graphic design
* Language tutoring ↔ Music lessons
